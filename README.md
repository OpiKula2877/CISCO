# Cisco — CCNA1: Příprava na závěrečnou zkoušku (2026)

Tento web je studijní dokument zaměřený na přípravu na závěrečný test z kurzu CCNA1 pro rok 2026.

- **Hostováno na:** https://cisco.opikula.dev
- **Hlavní stránka:** [index.html](index.html)

O čem je stránka:

Stručný přehled a shrnutí klíčových témat CCNA1, praktické příklady, cvičení a odkazy na další materiály. Materiál je určený jako souhrnná příprava před finálním testem.

Hlavní kapitoly (přibližně):

1. Úvod do sítí a základní pojmy
2. Modely TCP/IP a OSI
3. Ethernet, přepínání a VLAN
4. IP adresování a subnetting (IPv4 a základy IPv6)
5. Základy směrování (static, dynamic basics)
6. Základy WAN technologií
7. Zabezpečení sítí a základní politika přístupu
8. Troubleshooting a diagnostika sítí
9. Příprava na závěrečný test — tipy a praktická cvičení

Stažení webové aplikace / PWA:

- Doporučuji nainstalovat webovou aplikaci pro lepší offline přístup a pohodlné studium.
- Aplikaci lze stáhnout přímo z webu — v prohlížeči hledejte možnost instalace (ikonka "Install" nebo výzva pro přidání na plochu). Na tomto webu je také dostupná volba ke stažení v panelu pro vyhledávání (search bar) vpravo.
- Po instalaci získáte rychlý přístup, offline obsah (pokud je povolen service worker) a lepší mobilní zážitek.

Jak otevřít stránku lokálně:

1. Otevřít `index.html` přímo v prohlížeči nebo
2. Spustit jednoduchý server v kořenovém adresáři projektu:

```bash
python -m http.server 8000
# otevřít: http://127.0.0.1:8000/
```

Pokud chcete, můžu:

- doplnit konkrétní rozpis kapitol na základě obsahu `index.html` a `Tahak.md`,
- připravit skript pro lokální vývoj (`package.json`) nebo
- automatizovat nasazení na GitHub Pages (`gh-pages`).
